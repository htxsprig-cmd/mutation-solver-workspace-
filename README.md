# Mutation Solver Workspace

A lightweight browser-based workspace for testing mutation-name or mutation-ID lookups.

## Run locally

From this folder, start a local server:

Mac/Linux:
``` bash
cd C:\path\to\parent-folder
python -m http.server 8000
```
Windows(CMD Prompt):
```bash
cd C:\path\to\parent-folder
python -m http.server 8000
```

Then open http://127.0.0.1:8000/ in your browser.

This demo uses illustrative example data and is not medical advice.
